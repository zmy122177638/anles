## Format Module
格式化模块，再也不用担心数据格式不对了

### formatDate 时间格式化

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| - | - | - | - | - |
| data | new Date解析 | string\|number | 2020/10/27 | 必传 |
| format | 自选你想要的格式如 MM-DD HH:mm | string | YYYY-MM-DD HH:mm:ss | YYYY-MM-DD HH:mm:ss |

### formatNumber 数字格式化

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| - | - | - | - | - |
| data | new Date解析 | string\|number | 2020/10/27 | 必传 |
| format | 自选你想要的格式如 MM-DD HH:mm | string | YYYY-MM-DD HH:mm:ss | YYYY-MM-DD HH:mm:ss |
